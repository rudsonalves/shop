## Requisição HTTP

Usaremos comunicação via Rest API
Convenção: Endpoint HTTP (URL) + Método HTTP = Ação

GET -> Obter dados
Post -> Inserir Dados
Patch -> Atualizar Dados (atualiza parcialmente)
Put -> Substituir Dados (atualiza totalmente)
Delete -> Excluir Dados
